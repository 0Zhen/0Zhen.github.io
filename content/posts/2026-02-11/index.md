在馬達選型或規格制定初期，我們經常面臨這樣的問題：**已知輸入功率和電壓，如何計算所需電流？** 更具挑戰的是，當馬達尚未選定時，該如何估算功率因數？本文將從基礎公式出發，逐步建立完整的計算方法論，並提供實務規格制定的建議。

## 一、基礎理論：三相功率與電流

## 核心公式

三相交流電的功率關係式：

```ini
P = √3 × V_L × I_L × cos φ
```

反推電流：

```ini
I_L = P / (√3 × V_L × cos φ)
```

**符號說明：**

- `P`：功率 (W)
- `V_L`：線電壓 (V)
- `I_L`：線電流 (A)
- `cos φ`：功率因數
- `√3 ≈ 1.732`

## 簡化計算（380V 三相）

當電壓固定為 380V 時：

```ini
I_L = P / (658.16 × cos φ)
```

或更簡潔的形式：

```css
I (A) ≈ P (kW) / (0.658 × cos φ)
```

**計算範例：**

假設輸入功率 3 kW，功率因數 0.82：

```makefile
I = 3000 / (1.732 × 380 × 0.82)
I = 3000 / 539.5
I ≈ 5.56 A
```

## 二、功率因數：設計初期的估算方法

## 問題的本質

功率因數 (cos φ) 表示電力系統的效率，但在馬達尚未選定時，我們如何知道這個值？

答案是：**使用業界經驗值**。

## 按馬達功率分類

經過多年工程實踐，業界總結出以下經驗值：

馬達功率範圍 滿載功率因數 (cos φ) < 1 kW 0.65 ~ 0.75 **1 ~ 5 kW** **0.80 ~ 0.85** 5 ~ 15 kW 0.85 ~ 0.88 15 ~ 50 kW 0.87 ~ 0.90 > 50 kW 0.88 ~ 0.92

**關鍵洞察：** 馬達功率越大，功率因數越好。這是因為大型馬達的設計更優化，漏磁和銅損相對更小。

## 按負載狀況分類

同一台馬達在不同負載下，功率因數也會變化：

負載狀況 功率因數 說明 滿載運轉 0.80 ~ 0.85 設計最佳點 75% 負載 0.75 ~ 0.82 常見運轉區 50% 負載 0.65 ~ 0.75 效率下降 空載/輕載 0.20 ~ 0.40 極低效率

**設計建議：** 對於 3 kW 級馬達，建議使用 **cos φ = 0.82** 作為設計基準值。

## 不同運轉區域的功率因數

如果您有效率地圖，可以根據運轉區域細分：

- **高扭矩區（接近最大扭矩）**：cos φ = 0.75 ~ 0.85
- **額定點附近**：cos φ = 0.82 ~ 0.87
- **低扭矩區（輕載）**：cos φ = 0.60 ~ 0.75

## 三、從扭矩到電流：使用效率地圖計算

## 為什麼需要效率地圖？

當您需要精確計算特定工況下的電流時，僅有功率規格是不夠的。效率地圖提供了 **（轉速、扭矩、效率）** 的三維關係，讓我們能夠：

1. 從機械輸出推算電氣輸入
2. 驗證設計餘裕是否足夠
3. 確認最壞工況下的電流峰值

## 計算流程

**步驟一：從扭矩計算軸輸出功率**

完整公式：

```bash
P_out (W) = T (N·m) × ω (rad/s)
```

實用簡化公式：

```java
P_out (kW) = T (N·m) × n (rpm) / 9549
```

**步驟二：從效率地圖查找效率**

在效率地圖上找到對應的 (轉速, 扭矩) 點，讀取效率 η。

**步驟三：計算輸入功率**

```ini
P_in = P_out / η
```

**步驟四：選擇功率因數**

- 優先從效率地圖讀取（如有提供）
- 否則使用前面章節的經驗值

**步驟五：計算輸入電流**

```ini
I_L = P_in / (√3 × V_L × cos φ)
```

## 實際案例計算

**給定條件：**

- 轉速：800 rpm
- 扭矩：23 N·m
- 效率：89%
- 電壓：380V 三相
- 功率因數：0.80（估計值）

**計算過程：**

**1. 計算軸輸出功率**

```ini
P_out = 23 × 800 / 9549
P_out = 18,400 / 9549
P_out = 1.93 kW
```

**2. 計算輸入功率**

```ini
P_in = 1.93 / 0.89
P_in = 2.17 kW
```

**3. 計算輸入電流**

```ini
I_L = 2170 / (1.732 × 380 × 0.80)
I_L = 2170 / 526.5
I_L = 4.11 A
```

**結論：** 在此工作點下，輸入電流約為 **4.1 A**。

## 四、實務案例：3kW 馬達規格制定

## 設計情境

假設我們需要制定一份馬達規格書，要求如下：

- **輸入功率上限：3 kW**
- **最大扭矩：23 N·m**
- **電源：380V 三相**

## 設計餘裕的重要性

從前面的計算可知，實際運轉點（800 rpm, 23 N·m）的輸入功率為 2.17 kW。為什麼規格要定為 3 kW？

**設計餘裕 = (3.0–2.17) / 2.17 = 38%**

這個餘裕涵蓋了：

- ✅ 效率隨溫度、老化的變化（±5~10%）
- ✅ 瞬態過載需求（短時間 120~150% 負載）
- ✅ 不同工況的變化（不同轉速/扭矩組合）
- ✅ 供應商選型彈性（不同品牌效率差異）

**經驗法則：** 設計餘裕建議在 **20~40%** 之間。

## 規格書範例

項目 規格值 備註 額定輸入功率 3 kW (max) 連續運轉 電源電壓 380V AC ±10% 三相，50/60 Hz **額定電流** **≤ 5.6 A** @380V, cos φ ≥ 0.82 最大扭矩 23 N·m 功率因數 0.80 ~ 0.85 滿載時 最低效率 ≥ 85% 額定工作點
