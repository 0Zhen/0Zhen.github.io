**定義**

標準差是用來衡量數據分散程度的指標，數值越大代表數據越分散。

**計算公式**

σ = √[Σ(xi - μ)² / (n-1)]

**符號說明：**

- xi = 第 i 個數據
- μ = 平均值
- n = 樣本數

**計算步驟**

1. 計算所有數據的平均值 μ
2. 計算每個數據與平均值的差 (xi - μ)
3. 將每個差值平方
4. 將所有平方值加總
5. 除以 (樣本數-1)
6. 開根號

**實際例子**

測量5個零件長度：98, 100, 99, 101, 102 mm

1. 平均值 μ = (98+100+99+101+102)/5 =      100
2. 計算差值平方：

- (98-100)² = 4
- (100-100)² = 0
- (99-100)² = 1
- (101-100)² = 1
- (102-100)² = 4

1. 總和 = 10
2. 10/(5-1) = 2.5
3. σ = √2.5 ≈ 1.58 mm

**實務工具：** Excel 函數 =STDEV.S() 或統計軟體

**三、CPK (Process Capability Index) 製程能力指數**

**定義**

CPK 用來評估製程是否能穩定地生產符合規格的產品。

**計算公式**

**主公式：**

CPK = min(CPU, CPL)

**細部公式：**

- CPU (上限能力指數) = (USL - μ) / (3σ)
- CPL (下限能力指數) = (μ - LSL) / (3σ)

**符號說明：**

- USL = 規格上限 (Upper Specification Limit)
- LSL = 規格下限 (Lower Specification Limit)
- μ = 製程平均值
- σ = 製程標準差

**計算步驟**

1. **收集數據**：取得足夠樣本（建議至少30個）
2. **計算統計值**：計算平均值 μ 和標準差 σ
3. **確認規格**：明確 USL 和 LSL
4. **計算 CPU**：(USL - μ) / (3σ)
5. **計算 CPL**：(μ - LSL) / (3σ)
6. **取最小值**：CPK = min(CPU, CPL)

**實際例子**

**情境：**某製程規格為 100±10mm

- 規格上限 USL = 110mm
- 規格下限 LSL = 90mm
- 實測平均值 μ = 98mm
- 標準差 σ = 2mm

**計算：**

- CPU = (110-98)/(3×2) = 12/6 =      2.0
- CPL = (98-90)/(3×2) = 8/6 =      1.33
- CPK = min(2.0, 1.33) = **1.33**

**結論：**此製程 CPK = 1.33，剛好在合格邊緣。由於平均值偏離中心（98 vs 100），建議調整製程使平均值接近目標值 100mm，可提升 CPK。
